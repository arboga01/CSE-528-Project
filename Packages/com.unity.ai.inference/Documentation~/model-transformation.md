# Transform, optimize, and inspect a model

After you import a model in Unity, you can apply tools to make it faster, lighter, or more secure.

Use this information to serialize, quantize, and encrypt a model. You can also use built-in tools like the Model Asset Inspector to inspect and debug models.

Page| Description |
|----|-------------|
| [Serialize a model](xref:sentis-serialize-a-model) | Learn how to convert a model into a serialized format for storage and later use in Sentis. |
| [Quantize a model](xref:sentis-quantize-a-model) | Discover how to reduce the size and computational requirements of a model by quantizing its weights. |
| [Encrypt a model](xref:sentis-encrypt-a-model) | Understand how to protect your model by encrypting it for secure usage in runtime environments. |
| [Inspect a model](xref:sentis-inspect-a-model) | Learn how to inspect the structure and contents of a model to understand its layers, inputs, and outputs. |
| [Model Asset Inspector](xref:sentis-model-asset-inspector) | Use the Model Asset Inspector to view and manage model assets in Unity, ensuring compatibility with Sentis. |
